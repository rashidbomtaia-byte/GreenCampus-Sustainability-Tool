# GreenCampus-Sustainability-Tool
Project Idea

This project is a simple web tool that encourages students to practice eco-friendly habits such as recycling, saving electricity, and reducing plastic use.

How It Supports Sustainability

It helps students build daily green habits and raises awareness about protecting the environment on campus.

Technologies Used
	•	HTML
	•	CSS
	•	GitHub

How to Use
	1.	Open index.html in a browser.
	2.	Mark the eco-friendly actions you complete.

Future Improvements
	•	Add a points system
	•	Add user login
	•	Store progress data
	- Add a leaderboard for top eco-friendly students
